## TASK-1053: Strengthen IDEA template and arch analyze ingestion: reduce generic defaults
**Meta:** P1 | M | READY | Focus:yes | 1-code-reasoning | claude-code | docs/refinement/TEMPLATE.md, cli/src/main/ts/application/commands/analyze-command.ts

**Depends:** none

### Context

Signal from valentinlineiro/smartcart-os TASK-005 and IDEA-strengthen-idea-schema-and-think-ingestion: `arch analyze` detects IDEA files but defaults `size: M`, `class: 2-code-generation`, and `acs: [(path)]` when IDEA files omit structured metadata. It also does not surface new refinement drafts in `docs/INBOX.md` as pending promotion items.

The IDEA file contract is too loose: only `Source:` and a human-written `Decision` are clearly required. Everything else is inferred or defaulted. This makes promotion preparation speculative and noisy — the analyzer produces generic placeholder ACs instead of useful ones.

This is the first bug report from an external project using ARCH. The signal is real and actionable.

### Acceptance Criteria

- [ ] `docs/refinement/TEMPLATE.md` updated with required promotion metadata fields: title, problem, source, proposed outcome, candidate task class, candidate size, dependencies, validation hints.
  - `file: docs/refinement/TEMPLATE.md`

- [ ] `arch analyze` reads IDEA metadata fields explicitly. When `class` or `size` fields are present in the IDEA: uses them. When absent: reports "missing field" instead of defaulting.
  - `prose: verified by creating an IDEA with explicit class:1-code-reasoning and size:S — analyze uses those values`

- [ ] `arch analyze` writes `AWAITING_PROMOTION` to `docs/INBOX.md` when refinement drafts exist in `docs/refinement/` that have no Decision field. Does not write if Decision is already set.
  - `prose: verified by creating a bare IDEA file — AWAITING_PROMOTION appears in INBOX`

- [ ] When required IDEA metadata is absent: `arch analyze` reports missing fields explicitly (e.g. "IDEA-foo: missing candidate-class, candidate-size") instead of generating placeholder ACs.
  - `prose: verified by running arch analyze on an underspecified IDEA — missing fields reported`

- [ ] `npm test` passes.
  - `prose: 544+ tests pass`

- [ ] `arch review` passes.
  - `cmd: node cli/dist/index.js review`
