Five principles

01

Git is the operating system. All state lives in git. No external databases, no SaaS lock-in.

02

Context is a budget, not a default. Every task declares exactly what context it needs.

03

Humans propose, AI closes gaps. Ideas come from humans. AI flags dependencies and surfaces patterns.

04

Routing over monogamy. Different tasks require different CLIs. Use the right tool per task.

05

Kaizen is structural. Every session ends with the system knowing more than when it started.
