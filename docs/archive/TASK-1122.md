### Context

A task an autonomous agent cannot complete will be started, worked on indefinitely, and never closed. The stale task check emits STALE_TASK and reclaims focus — but the agent restarts the same task next iteration. Infinite loop.

Fix: track start count per task. After governance.maxTaskAttempts (default 3) starts without a close, auto-escalate to ANDON_HALT and remove the task from the eligible focus set.

The attempt count lives in the task Meta line: `**Attempts:** N`. Govern increments it on each FOCUS_ACQUIRED. On third acquisition without close, govern writes ANDON_HALT to NOTIFICATIONS.md and sets task status to BLOCKED.
