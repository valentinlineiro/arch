### Context

Source: IDEA-xs-close-path-completion.md

ADR-009 permits XS tasks to self-archive when DeterministicACVerifier returns pass. In practice the Hansei validator fires regardless of size, requiring non-empty Forward Action, non-placeholder Cost, and category other than [no-issue]. This makes closing a 5-minute XS task take 3-5 correction cycles.

Fix: when size is XS and all cmd predicates pass (DeterministicACVerifier returns all-pass), skip Hansei category/cost/forward-action validation. Require only that Severity and Category are present (any value). The close path is the same — just with reduced ceremony for the smallest tasks.
