### Context

Source: IDEA-context-injector-planned-artifact-projection.md

The context injector (BuildIndex) extracts artifact references from task description prose and surfaces them in the context index. When a task says "This task will produce ADR-001", the injector includes ADR-001 as a referenced artifact: even though it doesn't exist yet. Downstream agents read the context and treat ADR-001 as authoritative, causing hallucination and incorrect assumptions.

Fix: filter artifact references to only those that exist on disk at index time.
