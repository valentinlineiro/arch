### Context

Every IDEA in docs/refinement/ must declare which Mission-class it belongs to. Without this, arch review cannot determine whether the IDEA is autonomous-scope or human-gated — it would need to infer from prose. Inference is probabilistic. A field lookup is deterministic.

arch review checks every IDEA-*.md in docs/refinement/ for a Mission-class field. If absent: ADVISORY (not blocking — allows gradual migration). If present but not in mission.autonomousScope or mission.humanGated: WARN (unknown class). If present and in humanGated but Decision is not AWAITING_HUMAN: WARN (human-gated IDEA not awaiting human confirmation).

The check is purely structural: read field, look up in config array, compare to Decision field. No model call.
