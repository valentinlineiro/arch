## Schema
<!-- Canonical field definitions for the machine-readable-block below. -->
<!-- All tools reading METRICS.md MUST parse the ```json block tagged machine-readable-block. -->

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `schema_version` | string | yes | Semver of this schema (current: `"1.0"`) |
| `last_updated` | string | yes | ISO 8601 date of last write (`YYYY-MM-DD`) |
| `summary.velocity` | float | yes | Tasks completed / tasks planned (1.0 = 100%) |
| `summary.p50_cycle_time_hours` | int | yes | Median cycle time in hours across all tasks in period |
| `summary.p90_cycle_time_hours` | int | yes | 90th-percentile cycle time in hours |
| `summary.avg_cost_dollars` | float | yes | Average USD cost per task |
| `summary.review_fail_rate` | float | yes | Fraction of tasks that hit REVIEW_FAIL (0.0–1.0) |
| `summary.drift_count` | int | yes | Number of arch review drift violations in period |
| `summary.token_usage.total_input_tokens` | int | yes | Cumulative input tokens across all tasks in period |
| `summary.token_usage.total_output_tokens` | int | yes | Cumulative output tokens across all tasks in period |
| `summary.token_usage.avg_input_per_task` | int | yes | Average input tokens per task |
| `summary.token_usage.avg_output_per_task` | int | yes | Average output tokens per task |
| `summary.token_usage.cache_hit_rate` | float | no | Fraction of input tokens served from cache (0.0–1.0) |
| `tasks[].id` | string | yes | Task identifier (`TASK-NNN`) |
| `tasks[].size` | string | yes | Effort size: `XS`, `S`, `M`, or `L` |
| `tasks[].turns` | int | no | Agent request/response cycles to complete (see TASK-FORMAT) |
| `tasks[].cycle_time_hours` | int | yes | Wall-clock hours from IN_PROGRESS to DONE |
| `tasks[].cost` | float | yes | USD cost for this task |
| `tasks[].review_fail` | bool | yes | Whether the task hit a REVIEW_FAIL before DONE |
| `tasks[].tokens_in` | int | no | Input tokens consumed by this task |
| `tasks[].tokens_out` | int | no | Output tokens generated by this task |
