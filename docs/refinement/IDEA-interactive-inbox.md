# idea: Interactive inbox

- **Class:** 6-writing
- **Size:** M
- **Status:** draft

## Problem

Current inbox is read-only. Humans cannot act on tasks directly from the inbox, limiting human-in-the-loop autonomy.

## Proposed Solution

Make inbox interactive with action prompts:

1. Add action menu to inbox (e.g., promote, start, view)
2. Enable direct task promotion from inbox
3. Add keyboard navigation
